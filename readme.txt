*************Ce projet est une demonstration sur webPack **********
*************Structure du projet **********************************
on a un dossier Controler qui contient nos Controller 
et un dossier Modele qui contient nos Queries 
Le Dossier scr est recommande par webPack et a l'interieur un fichier index.js
 qui est le point d'entree de webPack 
 Le dossier dist est generer automatiquement 
 et contient un fichier main qui va servir de point de sortie et qui regroupe les 
 fichier js de notre application 
 Le fichier package.json qui contient les configuration de webPack 
 Pour lancer le projet il faut se pausitionner au niveau de la racine du projet 
 et lancer les commandes suivantes :npm dev en mode developpement 
                                    npm prod en mode production
Au niveau du dossier database se trouve la base de donnee 
db_sama_diaba le nom de la base
